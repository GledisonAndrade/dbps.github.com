## 🎯 Como Acessar a Nova Seção Pizza

1. **Abra o aplicativo**
2. **Menu superior**: Clique em **"Distribuição"** (ícone 🍰)
3. **Escolha o período**: 7, 14, 30, 90 dias ou personalizado
4. **Visualize**:
   - Gráfico de pizza interativo
   - Legenda com números e percentuais
   - Estatísticas por categoria

## 📲 Como Instalar como App

### Chrome/Edge (Windows/Mac/Linux):
1. Abra: `localhost/teste3` ou seu domínio
2. Clique no **ícone de instalação** (barra de endereço)
3. Clique em **"Instalar"**
4. Pronto! Abre como app no seu sistema

### Safari (iPhone/iPad):
1. Abra o site em Safari
2. Clique no **ícone de compartilhamento**
3. **"Adicionar à Tela Inicial"**
4. Nome: "DiabetesCare"
5. Pronto!

### Firefox (se disponível):
1. Clique no **ícone de menu** (≡)
2. **"Instalar aplicativo"**

## 🔄 Como Gerar Relatório com Pizza

1. **Acesse** a seção **"Relatórios"** (PDF)
2. **Escolha o período** (ou use os botões rápidos: 7, 14, 30, 90)
3. **Clique** em **"Gerar Relatório"**
4. Você verá:
   - Evolução glicêmica (linha)
   - **Distribuição de glicemia (pizza)** ← NOVO!
   - Análise por horário
   - Recomendações médicas
5. **Clique** em **"Baixar PDF Completo"**
6. PDF com **TUDO** incluído (sem erros de tema!)

## 🐞 Problemas? Soluções:

### "A pizza não aparece no PDF"
→ Visite a seção Distribuição antes de gerar relatório

### "Tema mudou e PDF ficou estranho"
→ Já foi corrigido! Use qualquer tema agora

### "Não consigo instalar"
→ Use Chrome, Edge ou Firefox (Safari via tela inicial)

### "Não funciona offline"
→ O Service Worker precisa estar registrado (reinicie a página)
